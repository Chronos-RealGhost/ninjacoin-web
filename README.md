# 🥷 NinjaCoin Mining Platform

![NinjaCoin Banner](https://img.shields.io/badge/Status-Operating-brightgreen?style=for-the-badge&logo=ninja)
![Tech Stack](https://img.shields.io/badge/Stack-Node.js%20%7C%20MongoDB%20%7C%20Express-blueviolet?style=for-the-badge)

Uma plataforma de mineração de criptomoedas de alta performance com estética cyberpunk e sistema de licenciamento avançado.

## ✨ Funcionalidades Premium

- 🚀 **Minerador Integrado:** Motor XMRig otimizado.
- 🔐 **Licenciamento Ninja:** Sistema de chaves (24h, 7d, 30d, Vitalícia) com bloqueio por IP (Hardware Lock).
- 🌩️ **Cloud Ready:** Integração total com MongoDB Atlas para persistência global.
- 💳 **Pagamentos:** Checkout integrado com Stripe.
- 🕸️ **Dashboard Futurista:** Interface web moderna e responsiva.
- 🛡️ **Seguranca:** Hash de senhas SHA-256 e proteção de endpoints.

## 🛠️ Configuração

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Configure as variáveis de ambiente no arquivo `.env`:
   ```env
   STRIPE_SECRET_KEY=sua_chave_aqui
   PORT=3000
   ```
3. Inicie o sistema:
   ```bash
   npm start
   ```
